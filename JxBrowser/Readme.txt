1. RELEASE NOTES
================================================================================

JxBrowser is a  cross-platform library  that  provides  an embedded  web browser
for Swing/JavaFX applications. JxBrowser allows embedding  the most popular 
browser engine — Chromium into your Java application.

To  see  the  functionality  of  JxBrowser  in  action  run  the  JxBrowser Demo
application available in JxBrowser distribution package.

API specification and examples that show how to use JxBrowser
functionality in Java applications are available in the distribution package.

2. GETTING STARTED
================================================================================

The JxBrowser package contains following:
 - JxBrowser library (/lib/jxbrowser.jar)
 - JxBrowser Chromium binaries for Windows (/lib/jxbrowser-win.jar)
 - JxBrowser Chromium binaries for Mac OS X 64-bit (/lib/jxbrowser-mac.jar)
 - JxBrowser Chromium binaries for Linux 32-bit (/lib/jxbrowser-linux32.jar)
 - JxBrowser Chromium binaries for Linux 64-bit (/lib/jxbrowser-linux64.jar)
 - JxBrowser API Specification (/doc/javadoc)
 - JxBrowser examples (/samples)
 - JxBrowser Demo application (/demo)
 - JxBrowser Licence Agreement (License agreement.txt)
 - This Readme file (Readme.txt)
 - Software and hardware requirements (Requirements.txt)
 - Chromium Webkit licence (GNU LESSER GENERAL PUBLIC LICENSE - 
 Version 2.1.txt and GNU LIBRARY GENERAL PUBLIC LICENSE - Version 2.txt)


The installation instruction is available at
https://jxbrowser.support.teamdev.com/support/solutions/articles/9000012860

5. RELINKING
================================================================================

It is possible to re-link JxBrowser with a modified webkit component via object 
files, as required by LGPL licence.
The instruction about
how to relink JxBrowser library with webkit is located in archive with the 
object files, available for download at:
http://cloud.teamdev.com/downloads/jxbrowser/relink/jxbrowser-X.X-relink.zip

Use the actual JxBrowser version instead of X.X. For example: 6.6. It is also 
possible to use a 3-digit version. Indicate the version in format X.X.X. 
For example 6.5.1.

6. SIGNED JARS
================================================================================

All JAR files in JxBrowser distribution package are not signed. If you need to
use JxBrowser JAR files signed with official TeamDev signature, then you can
download distribution package with signed JAR files from:
http://cloud.teamdev.com/downloads/jxbrowser/jxbrowser-X.X.X-cross-desktop-win_mac_linux-signed.zip

Use the actual JxBrowser version instead of X.X.X. For example: 6.6.1.
